# 记忆辅助 Skill (Learn-To-Memory)

> "把知识点截图进来，剩下的交给我。"

输入学习材料（截图、文档或文本），输出经过「理解-整理-编码-复习」深度加工的记忆策略。整合 **费曼技巧**、**组块化记忆法** 和 **林彼得记忆法** 三大方法论。

## 目录结构

- `SKILL.md`：技能入口、触发方式、全局约束
- `rules/`：三大记忆法核心方法论
  - `feynman-technique.md`：费曼技巧（以教代学、类比简化）
  - `chunking-method.md`：组块化记忆法（分类分组、层级化）
  - `lin-peter-method.md`：林彼得记忆法（解构—图像—记忆—强化）
- `commands/auto-memory.md`：完整执行流程与输出模板
- `README.md`：人工阅读说明

## 使用方式

触发指令 `/auto-memory`，或直接上传截图/文本要求记忆辅助。

## 使用场景

- **概念太抽象**：上传截图，输出大白话解释 + 生活类比
- **错题记不住**：上传错题，输出思维陷阱 + 记忆口诀
- **列表背不下来**：上传长列表，输出分组 + 荒诞串联故事

## 维护约定

- 入口信息只写在 `SKILL.md`
- 执行步骤只写在 `commands/`
- 记忆法判定标准只写在 `rules/`
- 不在多个文件重复维护同一条规则
