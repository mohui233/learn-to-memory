# 学习记忆教练 (Learn-To-Memory)

> "把知识点截图进来，剩下的交给我。"

输入学习材料（截图、文档或文本），输出经过「理解-整理-编码-复习」深度加工的记忆策略。整合 **费曼技巧 (Feynman)**、**组块化记忆法 (Chunking)** 和 **林彼得记忆法 (Lin-Peter)** 三大方法论，先让你真正理解，再帮你彻底记住。

## 使用方式

触发指令 `/auto-memory`，或直接上传截图/文本要求辅助。

## 使用场景

- **概念太抽象**：上传截图，输出大白话解释 + 生活类比
- **错题记不住**：上传错题，输出思维陷阱 + 记忆口诀
- **列表背不下来**：上传长列表，输出分组 + 荒诞串联故事

## 维护约定

- 入口信息只写在 `SKILL.md`
- 执行步骤只写在 `commands/`
- 记忆法判定标准只写在 `rules/`
- 不在多个文件重复维护同一条规则
